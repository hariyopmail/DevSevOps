6、操作系统
    操作系统就是将计算机硬件的一系列丑陋的接口封装起来，
    转化成优雅，便捷的接口，给到应用程序
    操作系统就是组织、协调、控制你的计算机硬件

    BIOS Baise input output system  # 计算机自带的一个小系统
